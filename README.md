
# Tic-Tac-Toe Game in Python

This Python script implements a simple text-based version of Tic-Tac-Toe.
Play against another player or against the computer!

## Usage
- Run `tic_tac_toe.py` in your Python environment to start the game.
- Follow the prompts to make moves in the Tic-Tac-Toe grid.

Enjoy playing and have fun!
